Semantische Elemente in HTML

Bedeutung Beispiele Quellen

Was sind Semantische Elemente

Semantische Elemente sind Tags die in einem Html code Verwendet werden um ihn barrierefreier zu machen und verständlicher zu machen.

Welche Semantische Elemente gibt es

  • <header>
  • <nav>
  • <section>
  • <aside>
  • <article>
  • <footer>

Beispiel Code

header

Der header ist eine Überschrift

<header><h1>Dies ist eine wichtige Überschrift</h1></header>

nav

Das nav Element definiert eine Reihe von Navigationslinks.

<nav>
<a href="/example/">Example</a>
</nav>

section

Section ist ein Element welches Themen Auseinander hält. Häufig mit Überschrift.

< section>
<h1>Dies kann die Überschrift des Themas sein</h1>
<article>Dein Themen Inhalt</article>
< /section>

aside

Das aside Element definiert Inhalte, die neben dem Inhalt stehen, in dem sie platziert sind (wie eine Seitenleiste).

<h4>Beispiel Überschrift</h4>
<p>Hier steht zusätzlicher Inhalt</p>
</aside>

article

Ein Article sollte für sich genommen Sinn ergeben und unabhängig vom Rest der Website verbreitet werden können.

< article>
<h3>Dies kann die Überschrift des Themas sein</h3>
<article>Dein Themen Inhalt</article>
< /article>

footer

Das Element footer definiert eine Fußzeile für ein Dokument oder einen Abschnitt.

<footer>
<p>Autor Kontaktinfo:
</footer>
