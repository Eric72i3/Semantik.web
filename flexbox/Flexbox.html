Was ist Flexbox?

Flexbox ist ein modernes CSS-Layoutsystem, mit dem Elemente flexibel und übersichtlich angeordnet werden können.

Zum Beispiel:

  • Nebeneinander
  • Untereinander
  • Zentriert

Flexible Ausrichtung

Mit Flexbox lassen sich einfach steuern:

  • Abstände
  • Reihenfolge
  • Ausrichtung

Ohne feste Größen oder komplizierte Berechnungen.

Ideal für moderne Layouts

Flexbox eignet sich besonders für:

  • Navigationen
  • Karten
  • Buttons
  • Responsive Designs

Diese passen sich automatisch an verschiedene Bildschirmgrößen an.

Beispiel Code

display: flex

Aktiviert Flexbox für einen Container.

.container {
display: flex;
}

flex-direction

Richtung: horizontal (row) oder vertikal (column).

.container {
flex-direction: row;
flex-direction: column;
}

justify-content

Ausrichtung auf der Hauptachse.

.container {
justify-content: center;
justify-content: space-between;
}

align-items

Ausrichtung auf der Querachse.

.container {
align-items: center;
align-items: flex-start;
}

gap

Abstand zwischen Elementen.

.container {
gap: 10px;
gap: 20px 10px;
}

flex-wrap

Umbruch in neue Zeilen.

.container {
flex-wrap: wrap;
flex-wrap: nowrap;
}
